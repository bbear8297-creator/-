# 🎮 多快捷键宏控制器

一个 Windows 全局宏工具，可为不同的操作绑定不同热键，按下热键即开始执行，再次按下停止。  
使用 DirectInput 模拟，适合游戏等需要后台按键的场景。

## ✨ 功能
- 支持多个宏，每个宏绑定一个独立热键（如 F1、F2）
- 可设置有限次循环或无限循环
- 动作可自定义：鼠标点击、按键、输入文字
- 每个动作的延迟可精确到毫秒（如 30ms）
- 程序关闭自动保存配置，再次打开自动恢复
- 界面分为“操作台”和“编辑器”，清晰易用


点击“新建”，输入宏的名字（如“攻击”）、热键（如 f1）、循环次数（0 表示无限）。

在 JSON 编辑框中编写动作序列，例如：

json
[
  {"type": "click", "button": "left", "delay": 0.03},
  {"type": "press", "key": "enter", "delay": 0.03}
]
点击“保存此宏”。

切换到“操作台”标签页，点击“启动全部热键监听”。

现在按下你设置的热键（如 F1），宏就会运行；再按一次热键就会停止。

📦 自己打包
bash
pip install pyinstaller
pyinstaller --onefile --windowed --uac-admin --name="宏控制器" 宏控制器.py
打包后的 exe 在 dist 文件夹里。

⚠️ 常见问题
需要以管理员身份运行，否则全局热键可能无效。

不支持直接输入中文，如需输入中文请用剪贴板方式。

部分游戏反作弊系统可能拦截模拟按键，本工具无法保证 100% 有效。
